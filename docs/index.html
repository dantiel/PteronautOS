ExpressLRS fork · ESP8285 · CRSF protocol

01 Essência

O PteronautOS é um fork do ExpressLRS transformado em controlador de voo para ornitóptero.

Não é um quad — não há motores, nem ESC, nem Betaflight. O movimento é gerado por forma de onda que pilota dois servos de asa e um servo de leme.

CRSF (rádio) → rx_main → Ornithopter → waveform → PWM servos

02 Arquitectura

Ornithopter
Mixer, waveform generation, channel filtering
Zephyrus
MPU6050 gyroscope + Mahony AHRS + dual PID
ServoOutput
PWM synchronized to CRSF tick
Target
ESP8285 2400MHz RX

03 Zephyrus Gyro Module

Estabilização do leme de crista cefálica via MPU6050. Mahony AHRS para estimativa de atitude, dual PID para correção de roll angle e yaw rate.

Sensor MPU6050 (GY-521)
Interface I2C directo (GPIO4/5)
AHRS Mahony (registo a registo, ~1kHz)
PID Duplo: roll angle + yaw rate
Output Correcção ao leme de crista
RAM 62%
Flash 56%
