Форк ExpressLRS · ESP8285 · Протокол CRSF

01 Суть

PteronautOS — это форк ExpressLRS, превращённый в полётный контроллер для орнитоптеров.

Не квадрокоптер — ни моторов, ни ESC, ни Betaflight. Движение создаётся формой волны, управляющей двумя сервоприводами крыльев и сервоприводом руля.

CRSF (радио) → rx_main → Ornithopter → волна → ШИМ сервоприводы

02 Архитектура

Ornithopter
Микшер, генерация волны, фильтрация каналов
Zephyrus
Гироскоп MPU6050 + Mahony AHRS + двойной PID
ServoOutput
ШИМ, синхронизированный с тиком CRSF
Target
ESP8285 2400MHz RX

03 Гиромодуль Зефир

Стабилизация цефалического гребня через MPU6050. Mahony AHRS для оценки ориентации, двойной PID для коррекции угла крена и скорости рыскания.

Датчик MPU6050 (GY-521)
Интерфейс Прямой I2C (GPIO4/5)
AHRS Mahony (регистровый уровень, ~1кГц)
PID Двойной: угол крена + скорость рыскания
Выход Коррекция цефалического руля
RAM 62%
Flash 56%

pio run -e PteronautOS_ESP8285_2400_RX
