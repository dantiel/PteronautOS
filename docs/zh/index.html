ExpressLRS分支 · ESP8285 · CRSF协议

01 本质

PteronautOS是一个ExpressLRS分支，转化为扑翼机飞行控制器。

不是四轴飞行器——没有电机，没有ESC，没有Betaflight。运动由波形生成，驱动两个翼舵机和一个冠舵机。

CRSF（无线电）→ rx_main → Ornithopter → 波形 → PWM舵机

02 架构

Ornithopter
混控器、波形生成、通道滤波
Zephyrus
MPU6050陀螺仪 + Mahony AHRS + 双PID
ServoOutput
PWM同步于CRSF节拍
Target
ESP8285 2400MHz RX

03 西风神陀螺仪模块

通过MPU6050稳定头冠舵。Mahony AHRS用于姿态估计，双PID用于滚转角和偏航率校正。

传感器 MPU6050 (GY-521)
接口 直接I2C（GPIO4/5）
AHRS Mahony（寄存器级，约1kHz）
PID 双：滚转角 + 偏航率
输出 冠舵校正
RAM 62%
Flash 56%

04 构建 

pio run -e PteronautOS_ESP8285_2400_RX
