ESP8285 · CRSF protocol

01 सार

PteronautOS ExpressLRS पर निर्मित, ऑर्निथॉप्टर के लिए उड़ान नियंत्रक में रूपांतरित।

यह क्वाड नहीं है — कोई मोटर नहीं, कोई ESC नहीं, कोई Betaflight नहीं। गति तरंगरूप द्वारा उत्पन्न होती है जो दो पंख सर्वो और एक शिखा पतवार सर्वो को चलाती है।

CRSF (रेडियो) → rx_main → Ornithopter → तरंगरूप → PWM सर्वो

02 संरचना

Ornithopter
मिक्सर, तरंगरूप निर्माण, चैनल फ़िल्टरिंग
Zephyrus
MPU6050 जायरोस्कोप + Mahony AHRS + दोहरा PID
ServoOutput
PWM CRSF टिक के साथ समकालिक
Target
ESP8285 2400MHz RX

03 ज़ेफ़िरस जायरो मॉड्यूल

MPU6050 के माध्यम से शिखा पतवार स्थिरीकरण। Mahony AHRS अभिविन्यास अनुमान के लिए, रोल कोण और यॉ दर सुधार के लिए दोहरा PID।

सेंसर MPU6050 (GY-521)
इंटरफ़ेस प्रत्यक्ष I2C (GPIO4/5)
AHRS Mahony (रजिस्टर-स्तर, ~1kHz)
PID दोहरा: रोल कोण + यॉ दर
आउटपुट शिखा पतवार सुधार
RAM 62%
Flash 56%
