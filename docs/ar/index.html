ESP8285 · CRSF protocol

01 الجوهر

PteronautOS مبني على ExpressLRS، محوّل إلى متحكم طيران للطائرات الخفاقة.

ليست طائرة رباعية — لا محركات، ولا ESCs، ولا Betaflight. الحركة تُولَّد بواسطة شكل موجة تقود سيرفوَي جناح وسيرفو دفة قحفية.

CRSF (راديو) → rx_main → Ornithopter → شكل موجة → PWM سيرفوهات

02 المعمارية

Ornithopter
خلاط، توليد شكل موجة، تصفية القنوات
Zephyrus
MPU6050 جيروسكوب + Mahony AHRS + PID مزدوج
ServoOutput
PWM متزامن مع نبضة CRSF
Target
ESP8285 2400MHz RX

03 زيفيروس — وحدة الجيروسكوب

تثبيت دفة القحف عبر MPU6050. Mahony AHRS لتقدير الوضعية، PID مزدوج لتصحيح زاوية الانقلاب ومعدل الانعراج.

المستشعر MPU6050 (GY-521)
الواجهة I2C مباشر (GPIO1/GPIO3, v1.1)
AHRS Mahony (مستوى السجل، ~1kHz)
PID مزدوج: زاوية الانقلاب + معدل الانعراج
المخرج تصحيح دفة القحف
RAM 62%
Flash 56%
