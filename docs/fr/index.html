ESP8285 · Protocole CRSF

01 Essence

PteronautOS est construit sur ExpressLRS, transformé en contrôleur de vol pour ornithoptères.

Pas un quad — pas de moteurs, pas d'ESC, pas de Betaflight. Le mouvement est généré par des formes d'onde pilotant deux servos d'aile et un servo de gouvernail de crête.

CRSF (radio) → rx_main → Ornithopter → forme d'onde → servos PWM

02 Architecture

Mixeur, génération de forme d'onde, filtrage de canaux
Zephyrus

ServoOutput
PWM synchronisé au tick CRSF
Cible
ESP8285 2400MHz RX

03 Module Gyroscopique Zephyrus

Stabilisation du gouvernail de crête céphalique via MPU6050. Mahony AHRS pour l'estimation d'attitude, double PID pour correction d'angle de roulis et de taux de lacet.

Capteur MPU6050 (GY-521)
Interface I2C direct (GPIO4/5)
AHRS Mahony (niveau registre, ~1kHz)
PID Double : angle de roulis + taux de lacet
Sortie Correction du gouvernail de crête
RAM 62%
Flash 56%
