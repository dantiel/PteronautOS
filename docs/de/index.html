ESP8285 · CRSF-Protokoll

01 Essenz

PteronautOS ist auf ExpressLRS aufgebaut, umgewandelt in einen Flugcontroller für Ornithopter.

Kein Quad — keine Motoren, keine ESC, kein Betaflight. Die Bewegung wird durch Wellenformen erzeugt, die zwei Flügelservos und ein Kopfruderservo steuern.

CRSF (Funk) → rx_main → Ornithopter → Wellenform → PWM Servos

03 Zephyrus Gyromodul

Stabilisierung des Kopfruders über MPU6050. Mahony AHRS zur Lagebestimmung, dualer PID zur Korrektur von Rollwinkel und Gierrate.

Sensor MPU6050 (GY-521)
Schnittstelle Direkter I2C (GPIO4/5)
AHRS Mahony (Registerebene, ~1kHz)
PID Dual: Rollwinkel + Gierrate
Ausgabe Kopfruderkorrektur
RAM 62%
Flash 56%
