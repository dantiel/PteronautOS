ESP8285 · CRSFプロトコル

01 本質

PteronautOSはExpressLRSを基盤として、オーニソプター用フライトコントローラーに進化させたものです。

クアッドではありません — モーターもESCもBetaflightもなし。動作は波形によって生成され、2つの翼サーボとクレストラダーサーボを駆動します。

CRSF（無線）→ rx_main → Ornithopter → 波形 → PWMサーボ

02 構造

ミキサー、波形生成、チャンネルフィルタリング
Zephyrus
MPU6050ジャイロ + Mahony AHRS + デュアルPID
ServoOutput
CRSFティックに同期したPWM
Target
ESP8285 2400MHz RX

03 ゼピュロス ジャイロモジュール

MPU6050による頭部クレストラダーの安定化。Mahony AHRSで姿勢推定、デュアルPIDでロール角とヨーレートを補正。

センサー MPU6050 (GY-521)
インターフェース 直接I2C（GPIO4/5）
AHRS Mahony（レジスタレベル、約1kHz）
PID デュアル：ロール角 + ヨーレート
出力 クレストラダー補正
RAM 62%
Flash 56%

pio run -e PteronautOS_ESP8285_2400_RX
