Fork de ExpressLRS · ESP8285 · Protocolo CRSF

01 Esencia

PteronautOS es un fork de ExpressLRS transformado en controlador de vuelo para ornitópteros.

No es un quad — sin motores, sin ESC, sin Betaflight. El movimiento se genera mediante formas de onda que pilotan dos servos de ala y un servo de timón de cresta.

CRSF (radio) → rx_main → Ornithopter → forma de onda → PWM servos

02 Arquitectura

Ornithopter
Mezclador, generación de onda, filtrado de canales
Zephyrus
Giróscopo MPU6050 + Mahony AHRS + PID dual
ServoOutput
PWM sincronizado al tick CRSF
Target
ESP8285 2400MHz RX

03 Módulo Giroscópico Céfiro

Estabilización del timón de cresta cefálica mediante MPU6050. Mahony AHRS para estimación de actitud, PID dual para corrección de ángulo de balanceo y velocidad de guiñada.

Sensor MPU6050 (GY-521)
Interfaz I2C directo (GPIO4/5)
AHRS Mahony (nivel de registro, ~1kHz)
PID Dual: ángulo de balanceo + velocidad de guiñada
Salida Corrección al timón de cresta
RAM 62%
Flash 56%
