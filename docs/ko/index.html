ESP8285 · CRSF 프로토콜

01 본질

PteronautOS는 ExpressLRS를 기반으로 오니톱터용 비행 컨트롤러로 진화했습니다.

쿠드가 아닙니다 — 모터도, ESC도, Betaflight도 없습니다. 움직임은 파형에 의해 생성되며, 두 개의 날개 서보와 크레스트 러더 서보를 구동합니다.

CRSF (무선) → rx_main → Ornithopter → 파형 → PWM 서보

02 구조

Ornithopter
뱁서, 파형 생성, 채널 필터링
Zephyrus
MPU6050 자이로 + Mahony AHRS + 듀얼 PID
ServoOutput
CRSF 틱에 동기화된 PWM
Target
ESP8285 2400MHz RX

03 제피로스 자이로 모듈

MPU6050을 통한 두부 크레스트 러더 안정화. Mahony AHRS로 자세 추정, 듀얼 PID로 롤 각도와 요 레이트 보정.

센서 MPU6050 (GY-521)
인터페이스 직접 I2C (GPIO4/5)
AHRS Mahony (레지스터 레벨, ~1kHz)
PID 듀얼: 롤 각도 + 요 레이트
출력 크레스트 러더 보정
RAM 62%
Flash 56%
